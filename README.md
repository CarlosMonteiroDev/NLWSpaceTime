## 🖥️ Projeto

Esse é um projeto responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnoligas

Esse projeto foi desenvolvido durante a NLW Spacetime da Rocketseat com as seguintes tecnologias:

    - HTML
    - CSS
    - GIT e Github
